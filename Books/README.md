## What's Inside

### Part I: From Data Birth to Digital Era
- Evolution of data in construction
- Data silos and integration challenges
- Modern management systems

### Part II: Data Types, Systems and Tools
- Structured data (Excel, SQL, CSV)
- Semi-structured data (IFC, JSON, XML)
- Unstructured data (PDF, photos, scans)
- Pandas & LLM for data automation

### Part III: Data-Based Calculations
- Cost estimation methodologies
- Quantity Take-Off (QTO) automation
- 4D-8D BIM and CO2 calculation
- ERP system integration

### Part IV: Analytics, Automation and ML
- KPI dashboards and visualization
- ETL pipeline automation
- BIM validation pipelines
- Machine learning for predictions

### Chapter 5: Threats and Strategy
- Digital transformation roadmap
- Open data vs proprietary systems
- Future of construction data
